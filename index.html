How the module works

There are two components - an HTML widget, and a raw JSON stream. The HTML widget can be dropped into a page wherever it's needed and should 'just work'. The JSON stream is just the raw image data and gives you extra control over presentation.

Script is initialised as in the following codeblock with an options object as a parameter:

 	$(document).ready(function(){
 		$.flickrModule.FlickrPhotosSearch({
 			hook: '.flickrBits',
 			per_page: 12,
 			term: 'portrait'
 		});
 	});

This zip package contains a "bootstrap" folder. It contains Twitter Bootstrap files and is used for presentation only.

All relevant code (aside from the initialisation calls in index.html) can be found in js/module.flickr.js
